# Caffy - POS & Offline Digital Menu ☕🚀

Caffy is a modern, robust, and 100% offline Point of Sale (POS) system designed specifically for coffee shops and local restaurants. Built to overcome internet connectivity issues, it operates entirely on a local Wi-Fi network.

## Key Features:

- 📱 **QR Digital Menu**: Customers can order directly from their smartphones using the local network.
- 👨‍🍳 **Real-Time Kitchen Monitor**: Instant order syncing via WebSockets (No page reloads needed).
- 📦 **Smart Inventory**: Recipe-based automatic stock deduction.
- 🔒 **HWID Security**: Hardware-locked licensing system for ultimate anti-piracy protection.
- ⚡ **Zero Internet Required**: Operates flawlessly during internet outages.

> [!NOTE]
> This repository contains the promotional Landing Page for the software. The core system is distributed as a compiled binary for maximum security.
